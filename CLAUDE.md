\# 교육청AI시스템 운영 매뉴얼



\## 1. 시스템 목적

이 폴더는 강원특별자치도교육청 업무를 AI와 함께 체계적으로 운영하기 위한 중심 폴더이다.



Claude Code, Codex, ChatGPT가 이 폴더 구조와 규칙을 기준으로 업무를 지원한다.



\## 2. 기본 원칙

\- 사용자가 초보자일 수 있으므로 항상 한 단계씩 안내한다.

\- 기존 파일은 임의로 삭제하지 않는다.

\- 원본 자료는 보존하고, 수정본은 새 파일로 만든다.

\- 큰 용량 파일은 GitHub에 올리지 않는다.

\- 업무별 자료는 반드시 해당 폴더에 저장한다.

\- 결과물은 한글 업무 문서에 바로 붙여 넣을 수 있게 작성한다.



\## 3. 폴더 역할



\### 00\_AI시스템

AI 시스템 설명서, 운영 규칙, Claude/Codex 공통 안내 파일을 저장한다.



\### 01\_예산관리

예산현황, 집행내역, 정산자료, 예산 자동화 프로그램을 저장한다.



\### 02\_공문서

공문, 시행문, 운영계획서, 결과보고서, 안내문 템플릿을 저장한다.



\### 03\_행사운영

하모니 페스티벌, 강원학생예술실기대회, 180초 영화제 등 행사 자료를 저장한다.



\### 04\_국제교류

일본 돗토리현 교류, 국제교육원 협의, 번역자료, 방문계획을 저장한다.



\### 05\_보도자료

보도자료, 카드뉴스 문안, 홍보자료, 언론 대응자료를 저장한다.



\### 06\_공동스킬

Claude, Codex, ChatGPT에서 함께 사용할 공통 프롬프트와 업무 스킬을 저장한다.



\### 07\_자동화

Python 프로그램, 자동화 스크립트, 실행 파일, 테스트 파일을 저장한다.



\### 08\_백업

중요 자료의 백업본을 저장한다.



\## 4. AI 작업 규칙

\- 사용자가 “다음”이라고 하면 현재 단계 다음 작업만 안내한다.

\- 사용자가 “초보자처럼 설명해줘”라고 하면 명령어 하나씩 설명한다.

\- 사용자가 파일을 만들라고 하면 저장 위치와 파일명을 먼저 확인한다.

\- 업무 문서는 공공기관 문체로 작성한다.

\- 보도자료는 강원특별자치도교육청 보도자료 형식을 우선 적용한다.

\- 예산 관련 작업은 금액, 합계, 집행률, 확인 필요 사항을 분리해서 정리한다.

\- 불확실한 내용은 임의로 채우지 말고 “확인 필요”라고 표시한다.



\## 5. GitHub 사용 규칙

\- 작은 설정 파일과 코드 파일만 GitHub에 올린다.

\- 대용량 한글, PDF, 엑셀, 이미지 파일은 원칙적으로 GitHub에 올리지 않는다.

\- 작업 후에는 다음 순서로 저장한다.
