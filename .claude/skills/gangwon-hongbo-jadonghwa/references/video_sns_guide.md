# 영상·SNS 콘텐츠 작성 가이드

카드뉴스 문구와 톤을 맞춘다. 영상은 **자막 중심(무내레이션 + 무료 BGM)** 을
기본으로 한다. SNS에서 무음+자막 시청이 많아 오히려 문법에 맞고, 제작 비용이 0.

## 릴스 대본 (15초)

- 초 단위 표로: `초 | 화면 | 자막`
- 카드뉴스 감성형 핵심 3~4컷을 뽑아 15초에 압축
- 마지막 1~2초는 로고 + 행사명

예시 골격:
| 초 | 화면 | 자막 |
|----|------|------|
| 0~2s | 무대 조명 켜짐 | (훅 한마디) |
| 2~5s | 부문 클로즈업 | (핵심 이미지) |
| 5~11s | 하이라이트 | (행사명·기간·장소) |
| 11~15s | 로고 | (마무리 + BI) |

## 쇼츠 대본 (30초)

- 부문이 여럿이면 부문별 4~5초씩 배분
- 각 부문 전환 시 BI 컬러 슬라이드 0.5초 삽입
- 구성: 훅(0~3s) → 부문 순회 → 하이라이트 몽타주 → 로고·문의

## 썸네일 문구 (3안)

- 큰 글씨 1줄 + 작은 글씨 1줄 구조
- 배경은 무대 사진 + BI 컬러 오버레이
- 서로 다른 각도로 3안 제시(감정형/규모형/부문형 등)

## SNS 게시글 (인스타·페이스북 캡션)

- 첫 줄에 핵심(행사명+한 줄 매력), 이어서 일정·부문·장소
- 부문은 이모지 불릿으로 가독성 확보(🎤🎼🎻🎶)
- 마지막에 장소·문의(부서·대표번호)
- 과한 이모지·해시태그 남발 금지. 담백하게.

## 해시태그

- 기관·행사 고정 태그 + 주제 태그 + 부문 태그를 섞어 12~16개
- 예: #강원교육 #강원특별자치도교육청 #(행사명) #학교예술교육
  #(부문들) #(지역/장소) 등
- 특수문자·띄어쓰기 없이 작성.

## 저장·게시 원칙

- 산출물은 폴더로 정리(카드뉴스/영상/대본·SNS/원본).
- **자동 게시는 하지 않는다.** 업로드용 폴더 저장까지가 안전선이며,
  실제 인스타·유튜브 게시는 담당자가 직접 한다(보안·감사).
