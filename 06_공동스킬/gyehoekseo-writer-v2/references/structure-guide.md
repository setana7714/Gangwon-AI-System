# 계획서 문서 구조 상세 가이드 (2026 신규 BI 양식 · 2026. 7. 서체 확정판)

## 1. 표지 레이아웃

```
┌─────────────────────────────────────────────┐
│ [BI 심볼]              [모두가 빛나는       │  ← 양식 고정, 수정 불필요
│                          강원교육 로고타입] │
│                                             │
│ ━━━─────────────────────── ┐                │
│            [계획서 제목]    │← 제목 문단     │
│ ───━━━─────────────────── ┘  위아래 테두리  │
│                                             │
│              2026.  ○.                      │
│                                             │
│              [원형 엠블럼]                  │
│           강원특별자치도교육청              │
│            문화체육특수교육과               │
└─────────────────────────────────────────────┘
```

- 제목 이중선 = **제목 문단의 위/아래 테두리** (위: 굵고얇게 THICK_SLIM,
  아래: 얇고굵게 SLIM_THICK, 네이비 0.5mm, 문단 테두리 간격 8mm)
- 조정법: 제목 문단에 커서 → [문단 모양] → [테두리/배경] 탭에서
  선 종류·굵기·색·간격 자유 변경. 문단이 지워지지 않는 한 선 유지.
- 색상: 네이비 `#005078`, BI 블루 `#1BA8E1`
- 표지에는 쪽번호 없음 (양식에 내장)

## 2. 본문(2쪽~) 레이아웃

```
일상이 예술이 되는 교실, 예술로 깊어지는 강원교육   ← 슬로건(고정)

━━━───────────────────────────
              [계획서 제목]      ← 22pt 네이비, 위아래 테두리(간격 4mm)
───━━━─────────────────────────
              강원특별자치도교육청 문화체육특수교육과 ← 오른쪽 정렬

[Ⅰ] 근 거          ← 네이비 뱃지 + 16pt 네이비 제목 + 블루 밑줄 0.4mm
✔ 내용              ← 휴먼명조 13pt 보통, 문단 위 10
□ 내용              ← 헤드라인M 14pt, 문단 위 15
  ○ 내용            ← 휴먼명조 13pt 보통, 문단 위 10
    ― 내용          ← 휴먼명조 13pt 보통, 문단 위 6
      ※ 내용        ← 휴먼명조 13pt 보통, 문단 위 3
```

- 본문 모든 문단 왼쪽 정렬
- 섹션 밑줄도 섹션 문단의 아래 테두리 — [문단 모양 → 테두리/배경]에서 조정
- 꼬리말: `- 쪽 -` 가운데 (2쪽부터)

## 3. 기호 위계·서식 (2026. 7. 확정)

| 단계 | 기호 | 폰트 | 크기 | 굵기 | 문단 위 | 들여쓰기 |
|------|------|------|------|------|--------|---------|
| 섹션 | Ⅰ~Ⅶ | HY헤드라인M | 16pt | 진하게 | 자동 | 0 |
