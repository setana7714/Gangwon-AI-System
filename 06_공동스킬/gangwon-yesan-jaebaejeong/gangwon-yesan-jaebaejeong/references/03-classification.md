# 03. 학교급 · 설립유형 구분

## 학교급
| 원자료 | 정리 |
|---|---|
| 초등학교, 초, 초교 | 초 |
| 중학교, 중 | 중 |
| 고등학교, 고 | 고 |
| 특수학교, 특수 | 특수 |
| 기관·단체 등 | 기타 |

- 순서: **초 → 중 → 고 → 특수 → 기타**
- 학교급이 자료에 없거나 불명확하면 추정하지 말고 검토표에 "학교급 확인 필요"로 표시.
- 시트명(예: `합창(중)`)이 그 시트 모든 행의 학교급을 뜻하지는 않는다. **행 단위로 실제 학교급을 확인**한다.

## 설립유형
- 구분: **국립 · 공립 · 사립 · 기타** (순서 동일)
- 국립학교도 별도 구분한다. **국립을 공립 합계에 포함하지 않는다.**
- 초등 합창 부문에 국립학교가 포함될 수 있으므로, 초등 합창 정리 시 국립 누락 여부를 반드시 확인한다.
- 설립유형이 명시돼 있으면 그대로 반영하되 국립·공립·사립으로 표기 통일.
- 비어 있거나 불명확하면 추정하지 말고 검토표에 "설립유형 확인 필요"로 표시.
- 설립유형별 합계는 국립·공립·사립·기타 각각 산출.
- 참고: 국립학교는 실무상 드물다. 자료에 국립 표기가 있을 때만 국립으로 분류하고, 없으면 억지로 만들지 않는다.
