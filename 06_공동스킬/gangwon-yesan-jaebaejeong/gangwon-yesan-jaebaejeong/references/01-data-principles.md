# 01. 자료 사용 원칙 (최우선)

## 반드시 지킬 것
- 엑셀 재배정 내역은 **사용자가 직접 제공한 자료만**을 기준으로 작성한다.
- 사용자가 제공하지 않은 학교명·지역청·학교급·설립유형·예산금액·사업명·선정여부를 임의로 추가하거나 추정하지 않는다.
- 인터넷 검색, 일반 지식, 과거 대화 내용, 유사 사업 자료, 다른 사업 자료를 근거로 새 데이터를 만들어 넣지 않는다.
- 기존 샘플 엑셀은 **서식·시트 구성·표 스타일·정리 방식 참고용**으로만 사용한다.
- 샘플 파일의 학교명·금액·지역청·사업명은 이번 작업 자료로 명시되지 않은 이상 새 결과물에 반영하지 않는다.
- 사용자가 제공한 예산 계획서·선정 현황·신청 현황·재배정 대상 자료를 최우선 기준으로 한다.

## 불명확할 때
- 자료에 없는 값은 임의로 채우지 않고 검토표에 **"자료 없음" / "확인 필요" / "사용자 확인 필요"**로 표시한다.
- 학교명·지역청·학교급·설립유형·금액 중 하나라도 불명확하면 추정 입력하지 않고 검토표에 남긴다.
- 확인 가능한 내용만 정리하고, 확인 불가능한 내용은 별도 검토사항으로 분리한다.
- 자료 간 내용이 서로 다르면 임의 판단하지 않고 원자료별 차이를 검토표에 표시한다.
- 최종 엑셀에는 사용자가 제공한 자료에 근거한 항목만 포함한다.

## 비밀번호·파일형식
- 비밀번호가 걸린 파일은 사용자가 알려준 `5381`로 열 수 있는지 확인한다.
- `.ozd`(정부 OZ Report 뷰어 파일)는 암호화 바이너리라 데이터 추출이 불가능하다. 같은 내용의 xlsx/pdf 또는 사용자 직접 입력 자료를 근거로 사용한다.

## 개인정보
- 자료에 포함된 학교명·교사명·연락처·이메일 등은 외부에 노출하지 않는다. 결과물·보고 요약에 불필요하게 원문을 나열하지 않는다.
