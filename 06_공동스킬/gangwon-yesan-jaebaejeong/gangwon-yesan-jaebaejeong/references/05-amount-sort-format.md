# 05. 예산 금액 · 정렬 · 서식 · 파일명

## 예산 금액
- 금액은 반드시 숫자 형식. 쉼표 포함·문자형 금액은 숫자로 변환.
- **"원"인지 "천원"인지 반드시 확인.** 샘플이 천원 단위면 머리글에 `지원금액(천원)` 또는 상단에 `단위: 천원` 명시.
- 합계는 직접 입력하지 않고 **수식(SUM/SUMIF/SUMIFS)**으로 계산.
- 총액과 세부내역 합계가 다르면 검토표에 오류로 표시.
- 팀 단위 예산과 학교별 예산이 혼재하면 확인 필요로 표시.
- 학교별 배정액이 없으면 임의 균등배분 금지. 사용자가 균등배분 기준을 명시한 경우에만 그 기준으로 계산.

## 정렬 순서
1. 지역청 순서(춘천→…→고성)
2. 학교급(초→중→고→특수→기타)
3. 설립유형(국립→공립→사립→기타)
4. 학교명 가나다순
5. 같은 학교 여러 부문 → 부문명 순 또는 원자료 순 유지

## 서식(공문서형 행정자료 스타일)
- 제목: 첫 행 굵게·가운데 정렬, 연도+사업명 포함(예: `2026 학교예술교육 활성화 지원사업 재배정 내역`).
- 단위: 제목 아래 또는 표 우측 상단.
- 표 머리글: 굵게·가운데. 금액 열: 천 단위 구분 `#,##0`.
- 합계 행: 굵게(옅은 강조색 허용).
- 과도한 병합 지양(검토 어려워짐). 열 너비는 내용이 보이게 조정.
- 인쇄: 가로 방향, 한 페이지 너비 맞춤 고려(`landscape_fit`).
- 기존 샘플의 색상·테두리·제목 위치·표 구성을 가능한 범위에서 참고.
- 폰트: 맑은 고딕 등 일관된 서체.
- 코드: `reallocation_lib.excel_helpers()`의 `title/header/cell/autofit/landscape_fit` 사용.

## 파일명
- 기본: `연도_사업명_재배정내역_정리본.xlsx`
- 예: `2026_학교예술교육활성화지원사업_학생예술활동동아리_재배정내역_정리본.xlsx`
- 검토·수정본: `연도_사업명_재배정내역_검토완료.xlsx`
