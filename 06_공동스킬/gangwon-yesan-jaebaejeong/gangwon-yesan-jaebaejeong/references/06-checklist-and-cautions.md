# 06. 검토표 · 작업 전 확인 · 반드시 주의할 사항

## 작업 전 확인 목록
사업명 · 연도 · 총예산 · 예산 단위(원/천원) · 지원 대상(학교/팀) 목록 · 학교별 지원금액 · 지역청명 · 학교급 · 설립유형 · 국립학교 포함 여부 · 초등 합창 부문 국립 포함 여부 · 제외 대상 여부 · 기존 샘플 시트 구조 · 최종 파일명
> 부족한 정보가 있어도 작업을 중단하지 않는다. 확실한 것부터 정리하고 불확실한 것은 검토표에 "확인 필요"로 표시.

## 검토표 항목 (반드시 포함)
결과 표기: **이상 없음 / 확인 필요 / 오류 / 수정 완료**
1. 사용자 제공 자료 외 내용 포함 여부
2. 총예산 = 재배정 총액 일치 여부
3. 세부내역 합계 = 지역청별 합계
4. 세부내역 합계 = 학교급별 합계
5. 세부내역 합계 = 설립유형별 합계
6. 국립학교 누락 여부
7. 초등 합창 부문 국립학교 포함 여부
8. 공립·사립·국립 구분 누락 여부
9. 지역청 표기 오류 여부
10. 지역청 순서 오류 여부
11. 없는 지역청 포함 여부
12. 사용자 제시 지역청 누락 여부
13. 같은 학교 중복 배정 여부
14. 지원금액 빈칸 여부
15. 지원금액 0원 여부
16. 금액 단위 혼용 여부
17. 원/천원 단위 혼동 여부
18. 수식 합계 오류 여부
19. 제외 대상 포함 여부 / 비고 확인 사항

> `reallocation_lib.build_full_checklist()`가 2·3·4·5·8·10·11·12·13·14·15·17을 자동 채운다.
> 나머지(1·6·7·9·16·18·19)는 자료 해석이 필요하므로 사람이 확인해 결과를 갱신한다.

## 반드시 주의할 사항 (실수 다발)
- **가. 제공 자료 외 추가 금지** — 없는 학교/지역청/금액/학교급/설립유형 임의 추가 금지, 샘플 데이터 복사 금지, 과거대화·일반지식 보완 금지.
- **나. 지역청 누락** — 제시 지역청이 빠지지 않았는지, 미제시 지역청이 추가되지 않았는지.
- **다. 지역청 순서** — 가나다·자동정렬 금지, 지정 순서 준수.
- **라. 국립학교 누락** — 국립을 공립으로 처리 금지. 초등 합창 국립 별도 확인.
- **마. 금액 단위** — 원/천원 혼용 여부. 500,000원과 500천원 의미 확인. 불명확 시 검토표.
- **바. 합계** — 지역청별·학교급별·설립유형별·세부 총합 모두 일치. 합계는 수식.
- **사. 중복 학교** — 중복 오류인지 여러 부문 선정인지 확인. 부문 선정이면 비고에 사유, 단순 중복이면 검토표.

## 2회 이상 검토(필수 절차)
1. **1차** — `build_full_checklist()` 자동 검증 → 검토표 시트 작성.
2. **2차** — `python scripts/recalc.py 결과.xlsx` 실행 → 수식 재계산, `#REF!/#DIV/0!/#VALUE!` 등 오류 0 확인. 오류가 있으면 수정 후 재실행.
3. 두 차례 모두 통과해야 산출물로 인정.
