# 강원교육청 예산 재배정 엑셀 스킬

## 0. 이 스킬의 최종 역할
단순 엑셀 정리가 아니라 **예산 재배정 행정자료의 정확성 검토 도구**다.
산출물은 세 조건을 동시에 충족해야 한다.
1. **보기 좋은 표** — 지역청별·학교급별·설립유형별·부문별로 한눈에 파악
2. **검증 가능한 표** — 총액·지역별·학교급별·설립유형별 합계가 서로 일치
3. **바로 쓰는 표** — 공문 첨부·내부 검토·예산 자료로 즉시 활용

## 1. 최우선 원칙 — 자료 사용 (가장 중요)
> 상세: `references/01-data-principles.md`
- 엑셀 내역은 **사용자가 직접 제공한 자료만** 기준으로 작성한다.
- 제공되지 않은 학교명·지역청·학교급·설립유형·금액·사업명·선정여부를 **임의로 추가·추정하지 않는다.**
- 인터넷 검색·일반지식·과거 대화·유사/다른 사업 자료로 데이터를 만들어 넣지 않는다.
- **샘플 엑셀은 서식·시트구성·표스타일 참고용일 뿐**, 그 안의 학교명·금액·사업명은 새 결과물에 복사하지 않는다.
- 자료에 없는 값은 채우지 말고 검토표에 **"자료 없음 / 확인 필요 / 사용자 확인 필요"**로 표시한다.
- 자료 간 내용이 다르면 임의 판단하지 말고 원자료별 차이를 검토표에 남긴다.

## 2. 작업 절차
1. **작업 전 확인** — `references/06-checklist-and-cautions.md`의 "작업 전 확인 목록" 점검. 부족한 정보가 있어도 중단하지 않고, 확실한 것부터 정리 + 불확실은 검토표에 표시.
2. **자료 로드** — 비밀번호가 걸린 파일은 `5381`로 열기 시도. `.ozd`(OZ Report)는 데이터 추출 불가이므로 xlsx/pdf/직접입력 자료를 근거로 삼는다.
3. **정규화** — 지역청 표준명·학교급·설립유형 통일. `references/02-region-order.md`, `references/03-classification.md`. `scripts/reallocation_lib.py`의 `normalize_*` 사용.
4. **정렬** — 지역청 지정순서 → 학교급(초·중·고·특수·기타) → 설립유형(국립·공립·사립·기타) → 학교명 가나다. `sort_detail()`.
5. **시트 생성** — `references/04-sheet-structure.md` 기준. 기본 구성: **재배정 총괄 · 지역청별 재배정 내역 · 학교급별 재배정 내역 · 설립유형별 재배정 내역 · 세부(부문별) 재배정 내역 · 검토표.** 부문이 여러 개면 세부내역을 부문별 시트로 분리(예: 합창(초)·합창(중)·오케스트라·아카펠라).
6. **금액·서식** — 합계는 반드시 **수식(SUM/SUMIF/COUNTIF)**, 직접 입력 금지. 단위(원/천원)를 머리글·상단에 명시. `references/05-amount-sort-format.md`.
7. **검토(2회 이상)** — ① `build_full_checklist()`로 자동 검증 → 검토표 시트 작성, ② `scripts/recalc.py`로 수식 재계산 후 오류 0 확인. `references/06-checklist-and-cautions.md`.
8. **보고 + 공문** — `references/07-report-and-gongmun.md` 형식으로 요약 보고. 결과 공문이 필요하면 **gongmun-writer 스킬**로 연계(재배정 총액·부문별 금액·대상 지역청 수를 넘김).

## 3. 스크립트
- `scripts/reallocation_lib.py` — 지역순서·정규화·정렬·검증·엑셀 서식 헬퍼. **자료를 생성하지 않고** 정리·검증만 한다.
- `scripts/recalc.py` — LibreOffice로 수식 재계산 및 오류(#REF!/#DIV/0! 등) 검출. 사용법: `python scripts/recalc.py 결과파일.xlsx`

## 4. 파일명 규칙
`연도_사업명_재배정내역_정리본.xlsx` (예: `2025_강원학생하모니페스티벌_참가지원비_재배정내역_정리본.xlsx`)
검토·수정본: `연도_사업명_재배정내역_검토완료.xlsx`

## 5. 절대 금지 / 반드시 확인 (요약)
- 제공 자료 외 내용 추가 금지 · 샘플 데이터 복사 금지
- 지역청: 사용자 미제시 지역청 추가 금지, 제시 지역청 누락 금지, **가나다순 아닌 지정 순서**
- 국립학교는 공립에 합산하지 않고 별도 집계
- 원/천원 단위 혼용 금지 · 합계는 수식 · 세부/지역/학교급/설립유형 총합 일치
