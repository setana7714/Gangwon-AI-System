# AGENTS.md — 예산 재배정 작업 지침 (Codex / 에이전트용)

이 저장소는 **강원교육청 예산 재배정 엑셀 정리·검토 스킬**입니다.

## 트리거
사용자 요청에 '재배정', '예산 재배정', '재배정 내역', '재배정 엑셀', '지역청별/학교별 배정',
'하모니 페스티벌 재배정', '예술향유지원 재배정', '180초 영화제 재배정', '검토표' 등이 나오면
**반드시 아래 절차를 따르세요.**

## 절차
1. **먼저 `SKILL.md`를 끝까지 읽으세요.** 모든 규칙의 진입점입니다.
2. 세부 규칙은 `references/01~07`을 필요할 때 읽으세요.
   - 01 자료 사용 원칙(최우선) · 02 지역청 순서 · 03 학교급/설립유형
   - 04 시트 구성 · 05 금액/정렬/서식/파일명 · 06 검토표/주의 · 07 보고/공문
3. 공용 로직은 `scripts/reallocation_lib.py`(정규화·정렬·검증·서식)를 사용하세요.
4. 수식 재계산·오류검출은 `python scripts/recalc.py <파일.xlsx>` (LibreOffice 필요).

## 절대 원칙 (요약)
- 사용자가 제공한 자료만 사용. 학교명·지역청·금액·학교급·설립유형·사업명을 **임의 생성·추정 금지**.
- 샘플 엑셀은 서식 참고용. **샘플 데이터를 결과물에 복사 금지.**
- 지역청은 지정 순서(춘천→…→고성). 없는 지역청 제외, 제시 지역청 누락 금지.
- 국립은 공립에 합산 금지.
- 합계는 수식. 세부/지역청별/학교급별/설립유형별 총합 일치. 원/천원 단위 확인.
- **반드시 2회 이상 검토**(자동 검증 + recalc)로 오류 0 확인.
- 결과 공문은 gongmun-writer 규칙(references/07)에 따라 작성.

## 환경
- Python 3, `openpyxl`, `pandas` 필요. 수식 재계산에 LibreOffice(`soffice`) 필요.
- 비밀번호 걸린 샘플: `5381`.
