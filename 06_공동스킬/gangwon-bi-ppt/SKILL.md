# 강원교육 BI PPT 스킬 (모두가 빛나는 강원교육)

## 역할
강원특별자치도교육청 **신규 BI(2026, '모두가 빛나는 강원교육')** 기반 발표자료를
생성·수정하는 전문 PPT 에이전트. BI 매뉴얼에서 벡터 추출한 투명 PNG 에셋과
18종 모델 슬라이드를 활용해 **어떤 내용이든 즉시 교육청 스타일 슬라이드로** 만든다.

## 워크플로우

### A. 새 PPT 생성
1. **반드시 `/mnt/skills/public/pptx/SKILL.md`를 먼저 읽는다.**
2. 입력(파일/주제) 분석 → 슬라이드 구성안 작성(기본 10장 내외, 요청 시 조정)
3. `scripts/generate_deck.js`를 복사해 내용만 교체하여 PptxGenJS로 생성
   - 슬라이드 크기: `defineLayout({name:"GW", width:13.333, height:7.5})` (16:9)
   - 에셋 경로는 스킬의 `assets/` 폴더 절대경로로 수정
4. 마무리 슬라이드에 회전 애니메이션 필요 시 → 아래 [회전 로고] 절차
5. **QA 필수**: pptx SKILL.md의 QA 절차(이미지 변환 → 검수)

### B. 기존 PPT 수정 / 슬라이드 추가
1. `assets/model_deck.pptx`(18종 모델)에서 필요한 슬라이드 디자인을 참조
2. 사용자가 준 PPT는 unpack → 슬라이드 XML 편집 or 모델 슬라이드 XML 이식 → pack
3. 텍스트만 바꿀 때는 XML 내 `<a:t>` 교체가 가장 안전

### C. 회전 로고(감사합니다) 슬라이드
1. PptxGenJS에서 심벌 이미지에 `objectName: "SPIN_SYMBOL"` 지정,
   이미지 파일은 **반드시 `assets/symbol_centered.png`** (회전 중심 보정본)
2. 생성 후 unpack → `python scripts/inject_spin.py <slideN.xml> SPIN_SYMBOL 8000` → pack
3. '감사합니다' 텍스트는 **별도 텍스트박스**로 아래에 배치(사용자가 삭제 가능해야 함)

## 모델 슬라이드 18종 (model_deck.pptx 순서)

1 표지A(화이트·엠블럼) · 2 표지B(네이비 임팩트) · 3 목차(5색 카드) · 4 섹션 간지
5 개조식 불릿+강조카드 · 6 2단 비교(현황→개선) · 7 핵심과제 3카드 · 8 4단계 프로세스
9 데이터 강조(수치 카드) · 10 표 스타일 · 11 연간 타임라인 · 12 5대 지향 꽃잎(편집형)
13 비전 체계도(원본+편집 알약) · 14 메시지 강조(워드마크) · 15·16 BI 에셋 라이브러리
17 마무리A(회전 심벌+감사합니다) · 18 마무리B(네이비 정적)

상세 좌표는 `references/design_spec.md`, 실제 코드는 `scripts/generate_deck.js` 참조.

## 필수 규칙
1. 컬러 값에 "#" 절대 금지 · 옵션 객체 재사용 금지(factory 사용) · `breakLine:true` 필수
2. 유니코드 불릿 문자 직접 입력 금지 → `bullet:{characterCode:"2219"}`
3. **로고 비율 변경·재채색·회전(장식 목적) 금지** — BI 규정. 배경은 흰색 또는 네이비(흰 원판 위 로고)만
4. 액센트 바·헤더 스트라이프 금지 — 헤더는 시그니처+타이틀 텍스트만으로 구성
5. 우측 여백 0.5in 이상 유지, 푸터 y=7.06 고정
6. 새 디자인 요소를 임의 창작하지 말 것 — BI 매뉴얼 범위(에셋+컬러+틴트+꽃잎 모티프) 안에서만
7. 표지·간지·마무리 외 본문 슬라이드에는 header()+footer() 패턴 유지
8. QA(이미지 변환 후 검수) 없이 납품 금지
