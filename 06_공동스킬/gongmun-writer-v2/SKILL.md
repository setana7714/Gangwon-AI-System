# 공문 작성 스킬 v2

## 목적

강원특별자치도교육청 문화체육특수교육과 업무 맥락에 맞는 공문 초안을 작성한다. 사용자가 완전한 자료를 주지 않아도 가능한 범위에서 먼저 초안을 만들고, 부족한 정보는 `[확인 필요]`로 표시한다.

공문 샘플 PDF 원본, 실제 문서번호, 학교명, 개인명, 연락처, 계좌, 주민등록번호, 내부 민감정보는 스킬에 저장하지 않는다. 샘플을 참고할 때는 구조와 문체 규칙만 추출한다.

## 지원 유형

1. 내부결재용 공문
2. 외부발송용 공문
3. 알림 공문
4. 제출 요청 공문
5. 선정 결과 통보 공문
6. 재배정 공문
7. 붙임 목록 작성

## 기본 workflow

1. 요청 의도를 공문 유형으로 분류한다.
2. 입력 자료에서 수신자, 제목, 근거, 핵심 내용, 기한, 제출 방법, 붙임 자료를 추출한다.
3. 부족한 정보는 질문을 최소화하고 `[확인 필요]`로 표시한다.
4. 기안용지 서식이 없으면 본문 초안부터 작성한다.
5. 공공기관 문체로 제목, 본문, 붙임, 끝 표시를 구성한다.
6. 발송 전 확인해야 할 항목을 마지막에 짧게 안내한다.

상세 문체와 형식은 `references/format-guide.md`를 읽고 따른다.

## 작성 원칙

- 문장은 간결하고 행정문서답게 작성한다.
- 사실, 날짜, 금액, 대상, 기한은 임의로 만들지 않는다.
- 확인되지 않은 정보는 반드시 `[확인 필요]`로 둔다.
- 사용자가 초안을 요청하면 질문을 많이 하지 말고 먼저 작성한다.
- 개인정보와 민감정보는 원문 그대로 재저장하지 않는다.
- 학교명이나 기관명이 불명확하면 `[학교명 확인 필요]`, `[기관명 확인 필요]`로 표시한다.
- 문서번호가 불명확하면 `[문서번호 확인 필요]`로 표시한다.
- 붙임 파일명이 불명확하면 `[붙임 파일명 확인 필요]`로 표시한다.

## 기본 출력 형식

공문 초안은 다음 순서로 출력한다.

```text
수신  [수신자]
(경유) [경유자, 없으면 생략]
제목  [공문 제목]

1. [근거 또는 배경]

2. [알림, 요청, 통보, 재배정 등 핵심 내용]

3. [기한, 제출 방법, 협조 사항 등]

붙임  1. [붙임명] 1부.
      2. [붙임명] 1부.  끝.
```

내부결재용은 수신란을 생략하거나 `내부결재`로 표시하고, 외부발송용은 수신 기관을 명확히 표시한다.

## 유형별 작성 기준

### 내부결재용

- 목적: 내부 검토와 결재를 위한 시행 전 보고
- 포함: 추진 근거, 주요 내용, 소요 예산, 향후 일정, 협조 사항
- 제목 예시: `[사업명] 추진 계획(안) 보고`

### 외부발송용

- 목적: 학교, 기관, 단체에 공식 안내 또는 요청
- 포함: 수신자, 제목, 근거, 요청 내용, 기한, 담당자 연락처
- 표현: `협조하여 주시기 바랍니다`, `제출하여 주시기 바랍니다`

### 알림 공문

- 목적: 일정, 행사, 회의, 연수, 결과 등을 안내
- 포함: 일시, 장소, 대상, 주요 내용, 참석 또는 확인 사항
- 표현: `다음과 같이 알려드리니 참고하여 주시기 바랍니다`

### 제출 요청 공문

- 목적: 자료, 명단, 결과보고서, 정산서류 등 제출 요청
- 포함: 제출 대상, 제출 자료, 제출 기한, 제출 방법, 문의처
- 표현: `기한 내 제출하여 주시기 바랍니다`

### 선정 결과 통보 공문

- 목적: 공모, 사업, 대상교, 대상자 선정 결과 안내
- 포함: 선정 결과, 후속 절차, 유의 사항, 붙임 명단
- 표현: `선정 결과를 다음과 같이 알려드리니`

### 재배정 공문

- 목적: 사업비 또는 예산을 학교·기관에 재배정
- 포함: 재배정 대상, 금액, 사업명, 집행 기준, 정산 또는 결과보고 기한
- 금액이 불명확하면 `[금액 확인 필요]`로 표시한다.
- 표현: `사업비를 다음과 같이 재배정하오니 목적에 맞게 집행하여 주시기 바랍니다`

## 붙임 작성

- 붙임이 1개이면 `붙임  [붙임명] 1부.  끝.`
- 붙임이 2개 이상이면 번호를 붙인다.
- 수량이 불명확하면 `1부` 대신 `[수량 확인 필요]`로 표시한다.
- 붙임명에 실제 개인정보가 포함되면 일반화한다.

## 마지막 확인 안내

초안 뒤에 다음 항목 중 필요한 것만 간단히 안내한다.

- `[문서번호 확인 필요]`
- `[수신자 확인 필요]`
- `[제출 기한 확인 필요]`
- `[붙임 파일명 확인 필요]`
- `[금액 확인 필요]`
- `[담당자 연락처 확인 필요]`
