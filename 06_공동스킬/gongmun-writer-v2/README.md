# gongmun-writer-v2

강원특별자치도교육청 문화체육특수교육과 업무용 공문 초안 작성 스킬이다.

## 사용 대상

- 내부결재용 공문
- 외부발송용 공문
- 알림 공문
- 제출 요청 공문
- 선정 결과 통보 공문
- 재배정 공문
- 붙임 목록 작성

## 기본 원칙

- 기안용지 서식이 없어도 본문 초안부터 작성한다.
- 확인되지 않은 정보는 `[확인 필요]`로 표시한다.
- 질문을 최소화하고 가능한 범위에서 먼저 초안을 작성한다.
- 공문 샘플 PDF 원본은 저장하지 않는다.
- 문서번호, 학교명, 개인정보, 민감정보는 원본 그대로 저장하지 않는다.
