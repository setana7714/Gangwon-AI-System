# 보도자료 문체·구성 가이드 (강원특별자치도교육청 공식 톤)

도교육청 보도자료 관례(문화체육특수교육과 기존 보도자료 분석 결과)와
공식 서식 작성 요령을 반영한 상세 기준.

---

## 0. 대원칙

- **배포 시점**: 보도자료 배포는 1일 1회 오전 10시. 행사 종료 후 배포 → **전 문장 과거형**
- **분량**: 1쪽 기본(최대 2쪽), 문단 3~5개, 문단당 1~3문장
- **문체**: 개조식 아님. 신문 기사체(간결한 서술형 종결)
- **어깨 힘 빼기**: 수식어 남발 금지. 숫자·고유명사·프로그램명이 힘을 만든다

---

## 1. 표제 (승부처 — 기사 전파의 절반)

- 형식: `□ {표제} □` (기호는 스크립트 자동)
- 공식: **행사명 + 핵심 동사(개최/성료)** 또는 **핵심 메시지 + 행사명**
- 좋은 예
  - `2026 강원학생 하모니 페스티벌 성황리 개최`
  - `180초 안에 담긴 진심이 스크린을 흔들었다! 제5회 강원청소년 180초 영화제 개최`
- 나쁜 예
  - `행사 개최 알림` (정보 없음)
  - `강원특별자치도교육청, 2026년도 학교문화예술교육 활성화 지원 사업의 일환으로...` (문장형·장황)

## 2. 부제

- 형식: `“{부제}”` (따옴표 자동)
- 표제가 못 담은 **규모·특징·의미**를 한 줄로: 참가 수치, 새 프로그램, 캐치프레이즈
- 예: `“35개교 928명… 합창·오케스트라·아카펠라, 예술로 하나 된 강원교육”`

## 3. 전문 (리드 — □ 1문단)

육하원칙을 한 문장에 담는 고정 공식:

```
강원특별자치도교육청(교육감 강삼영)은 {날짜}(요일)부터 {날짜}(요일)까지
{장소}에서 '{행사명}'을 개최했다.
```

- 필요 시 두 번째 문장으로 행사 취지 1문장 추가
- **"오는", "개최한다", "개최할 예정" 절대 금지** — 이미 끝난 행사다

## 4. 본문 (□ 3~4문단 + ◦ 세부)

권장 문단 흐름 (행사 성격에 따라 가감):

| 순서 | 내용 | 서술 예 |
|------|------|--------|
| ① 규모·구성 | 참가 인원·부문·일자별 운영 | "이번 행사에는 도내 35개교 928명이 참가했다." |
| (◦) | 부문별 세부 | "◦ 아카펠라·중등합창(7. 13.~14.), 오케스트라·초등합창(7. 15.~16.)" |
| ② 프로그램·특징 | 올해 새 요소, 체험·특별무대 | "…특별 축하무대가 함께 열려 관객의 호응을 얻었다." |
| ③ 시상·성과 | 수상 내역, 표창 | "우수 참가교에는 교육감상이 수여됐다." |
| ④ 인용문 | 교육감(또는 과장) 발언 | 아래 5. 형식 |
| ⑤ 의의·전망 | 마무리 + 향후 계획 | "…운영해 나갈 계획이다." |

- 숫자는 진하게 강조하지 않아도 됨(서식상 본문은 휴먼명조 통일)
- 학교명·작품명은 「 」 또는 ' ' 로 표기

## 5. 인용문 형식

```
□ 강삼영 교육감은 "인용문 첫 문장. 인용문 둘째 문장."이라며
  "마무리 기대·당부 문장."이라고 말했다.
```

- 종결: 말했다 / 밝혔다 / 전했다 / 강조했다 (돌려쓰기)
- 인용문 내용: ① 행사의 교육적 의미 → ② 학생·교원 격려 → ③ 향후 지원 의지
- 과장 인용이 필요하면: `□ 이인범 문화체육특수교육과장은 "..."라고 밝혔다.`
- 인용문은 자료에 없으면 새로 작성하되, 행사 취지·교육감 교육철학
  (모두가 빛나는 강원교육)과 어긋나지 않게

## 6. 과거형 동사 사전

| 상황 | 과거형 표현 |
|------|-----------|
| 행사 개최 | 개최했다, 열렸다, 마련됐다, 진행됐다 |
| 참가 | 참가했다, 참여했다, 함께했다, 무대에 올랐다 |
| 운영 | 운영됐다, 구성됐다, 펼쳐졌다, 선보였다 |
| 시상 | 수여됐다, 시상했다, 선정됐다 |
| 반응 | 호응을 얻었다, 박수를 받았다, 눈길을 끌었다 |
| 발언 | 말했다, 밝혔다, 전했다, 강조했다 |
| (예외) 향후 | ~해 나갈 계획이다, ~하겠다고 밝혔다 |

## 7. 붙임

```
붙임  행사 사진 2매.  끝.
```

- 여러 개일 때:
  `붙임  1. 포스터 1매  2. 행사 사진 2매.  끝.`
- 사진 촬영 안내가 필요하면 "(행사 후 제공)" 부기 가능

## 8. 표기 세칙

- 날짜: `7. 13.(월)` — 월·일 뒤 마침표, 앞자리 0 생략, 요일 괄호
- 기간: `7. 13.(월)부터 16.(목)까지` 또는 `7. 13.(월)~16.(목)`
- 전화: `☎ 033) 258-5382`
- 행사명: 첫 등장 시 '2026 ○○○' 작은따옴표, 이후 "이번 행사/페스티벌/대회"
- 기관 첫 표기: `강원특별자치도교육청(교육감 강삼영)` — 이후 "도교육청"
