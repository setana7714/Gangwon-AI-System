# bodojaryo-writer — 강원특별자치도교육청 보도자료 HWPX 생성 스킬

강원특별자치도교육청 **2026 신규 BI 「모두가 빛나는 강원교육」** 공식 보도자료 서식으로
언론 배포용 보도자료를 **HWPX(한글) 파일**로 자동 생성하는 Claude 스킬입니다.

## 특징

- 공식 서식 완전 재현: BI 시그니처 로고 → 제호 표(보도자료/기관명·배포일·과장 연락처)
  → 자료문의(하단 괘선) → 표제 상자(연노랑 배경) → 전문·본문(□/◦) → 붙임
- **과거형 원칙 내장**: 보도자료는 행사 종료 후 기사화되므로 전 문장 과거형으로 작성
  ("개최했다", "수여됐다", "말했다") — 예정형 표현 자동 점검
- 표제 `□ ~ □`, 부제 `“ ”`, 본문 `□`/`◦` 기호 자동 부착
- 배포일 요일 자동 계산 (`2026. 7. 17.` → `2026. 7. 17.(금)`)
- 생성 파일은 한글(HWP)에서 표·배경·글자 모양까지 전부 수정 가능

## 폴더 구조

```
bodojaryo-writer/
├── SKILL.md                       # 스킬 본문 (트리거·규칙·체크리스트)
├── assets/
│   └── press-template.hwpx        # 신규 BI 보도자료 서식 (로고·표·표제상자 내장)
├── references/
│   └── writing-guide.md           # 문체·구성 상세 가이드 (표제 공식·인용문·과거형 사전)
└── scripts/
    └── build_press.py             # press.json → HWPX 생성 스크립트
```

## 사용법

Claude(claude.ai 또는 Claude Code)에 스킬 설치 후:

```
2026 강원학생 하모니 페스티벌 보도자료 써줘 (운영계획 첨부)
```

수동 실행:

```bash
python scripts/build_press.py press.json 보도자료.hwpx
```

`press.json` 예시:

```json
{
  "date": "2026. 7. 17.",
  "chief_line": "문화체육특수교육과장 : 이인범  ☎ 033) 258-5370",
  "inquiry": "자료문의 : ☎ 258-5382  문화교육담당 장학관 ○○○  담당자 ○○○",
  "headline": "2026 강원학생 하모니 페스티벌 성황리 개최",
  "subtitle": "35개교 928명, 나흘간 예술로 하나 된 강원교육",
  "paragraphs": [
    ["lv1", "강원특별자치도교육청(교육감 강삼영)은 ... 개최했다."],
    ["lv2", "부문별 세부 내용"]
  ],
  "attachment": "붙임  행사 사진 2매.  끝."
}
```

- `lv1` = □ 문단(휴먼명조 14pt), `lv2` = ◦ 문단(휴먼명조 13pt)
- 기호는 자동으로 붙으므로 텍스트에 포함하지 않습니다.

## 요구 사항

- Python 3.8+ (표준 라이브러리만 사용 — 외부 의존성 없음)
- 검증용(선택): `pip install python-hwpx`
